# Générateur de QR Code

Une application web simple pour générer des QR codes à partir de n'importe quel lien.

## Utilisation

1. Ouvre `index.html` dans un navigateur
2. Entre un lien dans le champ
3. Clique sur **Générer** ou appuie sur **Entrée**
4. Télécharge le QR code en PNG

![Aperçu](qrcode.png)

## Technologies

- HTML / CSS / JavaScript
- [QRCode.js](https://github.com/davidshimjs/qrcodejs)
